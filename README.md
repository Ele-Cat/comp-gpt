## 使用

- 下载

  ```
  git clone https://gitee.com/ele-cat/comp-gpt.git
  ```

- 安装

  ```
  yarn install
  ```

- 运行

  ```
  yarn serve
  ```

- 打包

  ```
  yarn build
  ```
